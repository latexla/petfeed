**Дата:** 2026-04-25

## Контекст

Текущий расчёт рациона использует упрощённую формулу RER × rer_multiplier из таблицы `nutrition_knowledge`. Он не учитывает статус кастрации, уровень активности, физиологический статус, возрастные группы щенков и нутриенты (белок, жиры, минералы). Фаза 1 исправляет это, закладывая фундамент для рекомендательной системы кормов (Фаза 2).

## Секция 1: Модель данных

### Новые поля в таблице `pets`

| Поле | Тип | Значения | Описание |
|---|---|---|---|
| `is_neutered` | bool | true/false | Кастрирован/стерилизован |
| `activity_level` | str | low/moderate/high/working | Уровень активности |
| `physio_status` | str | normal/pregnant/lactating/recovery | Физиологический статус |

`goal` остаётся (maintain/lose/gain/growth) — отвечает за цель, а не физиологию.

### Новая таблица `food_categories`

```sql
id          SERIAL PRIMARY KEY
name        VARCHAR(100)    -- "Сухой корм (стандарт)"
food_type   VARCHAR(50)     -- dry | wet | natural | raw
kcal_per_100g NUMERIC(6,2)
protein_pct NUMERIC(5,2)
fat_pct     NUMERIC(5,2)
fiber_pct   NUMERIC(5,2)
notes       TEXT
```

Начальные данные:

| Название | Тип | ккал/100г | Белок% | Жир% |
|---|---|---|---|---|
| Сухой корм (стандарт) | dry | 350 | 26 | 14 |
| Влажный корм (стандарт) | wet | 85 | 8 | 5 |
| Натуральный (мясо+овощи) | natural | 150 | 18 | 10 |
| BARF (сырое) | raw | 130 | 17 | 9 |

### Новая таблица `breed_risks`

```sql
id          SERIAL PRIMARY KEY
breed_name  VARCHAR(100)
risk_key    VARCHAR(50)  -- atopy | patellar_luxation | obesity | hypoglycemia_puppies
```

При создании питомца с известной породой → автозагрузка рисков. Риски влияют на рекомендации в выводе рациона.

### Реструктуризация стоп-листа

Текущее текстовое поле `stop_foods` в `nutrition_knowledge` заменяется таблицей `stop_foods`:

```sql
id               SERIAL PRIMARY KEY
species          VARCHAR(50)
level            INTEGER       -- 1 | 2 | 3
product_name     VARCHAR(200)
toxic_component  TEXT
clinical_effect  TEXT
source           VARCHAR(200)
```

- **Уровень 1** — универсальные токсины (всегда показываем)
- **Уровень 2** — нутриционально нежелательные (с пометкой)
- **Уровень 3** — аллергены породы (только информационно, без запрета — диагноз только у ветеринара)

## Секция 2: Логика расчёта

### MER-расчёт (NutritionService)

```python
RER = 70 × weight_kg ^ 0.75

# Базовый коэффициент по физиологическому статусу и возрасту
if age_months < 4:
    base = 3.0   # ⚠️ предупреждение о гипогликемии
elif age_months < 12:
    base = 2.0
elif physio_status == "pregnant":
    base = 2.5
elif physio_status == "lactating":
    base = 2.5
elif physio_status == "recovery":
    base = 1.3
elif is_neutered:
    base = 1.6
elif goal == "lose" or "obesity" in breed_risks:
    base = 1.4
else:
    base = 1.8   # интактный взрослый

# Поправка на активность
activity_multiplier = {
    "low": 0.8,
    "moderate": 1.0,
    "high": 1.2,
    "working": 1.6   # NRC фактор 180
}

MER = RER × base × activity_multiplier[activity_level]
```

### Нутриенты (AAFCO минимумы)

```python
is_puppy = age_months < 12 or physio_status in ("pregnant", "lactating")

protein_min_pct = 22.5 if is_puppy else 18.0   # % от СВ
fat_min_pct     = 8.5  if is_puppy else 5.5
ca_min_pct      = 1.2  if is_puppy else 0.5
p_min_pct       = 1.0  if is_puppy else 0.4

# Абсолютные значения в граммах
protein_g_per_day = (protein_min_pct / 100) × daily_food_grams
fat_g_per_day     = (fat_min_pct / 100) × daily_food_grams
```

### Кормлений в день (по возрасту, в коде)

| Возраст | Кормлений | Примечание |
|---|---|---|
| < 4 мес | 5 | ⚠️ Риск гипогликемии — не пропускать |
| 4–6 мес | 4 | |
| 6–12 мес | 3 | |
| 12+ мес | 2 | Утро и вечер |

### Формула нормы в граммах

```
daily_food_grams = MER ÷ kcal_per_100g × 100
food_per_meal    = daily_food_grams ÷ meals_per_day
```

## Секция 3: FSM бота

### Расширенный поток создания питомца

```
species → breed → name → age_unit → age → weight
  → neutered_status (пропуск если age < 12 мес)
  → activity_level
  → food_category
  → confirm
```

### Новые шаги

**Шаг 6 — Кастрация** (только для взрослых, age ≥ 12 мес)
```
Кастрирован/стерилизован?
[Да]  [Нет]
```

**Шаг 7 — Активность**
```
Уровень активности:
[Низкий]  [Умеренный]  [Высокий]  [Рабочий]
```

**Шаг 8 — Тип корма**
```
Чем кормите?
[Сухой корм]  [Влажный корм]
[Натуральный] [BARF (сырое)]
```

**Шаг 9 — Подтверждение**
```
Имя: Барсик      | Вид: Собака
Вес: 6 кг        | Возраст: 3 года
Кастрирован: Да  | Активность: Высокая
Корм: Сухой      | Статус: Норма

[Сохранить]  [Изменить]
```

Новые поля также редактируются в профиле питомца.

## Секция 4: Вывод рациона

### Основной экран рациона

```
Рацион для Барсика (6 кг, 3 года)

Энергия
  Калорий в день:  490 ккал
  Корма в день:    140 г  (сухой корм)
  Кормлений:       2 раза в день
  Порция за раз:   70 г

Нутриенты (минимум в день)
  Белок:  ~25 г
  Жир:    ~8 г
  Ca:P    соотношение 1.2–1.4:1

Рекомендации
  • Рекомендуется миска-лабиринт
  • При смене корма — переход 7–10 дней
  • Омега-3 добавки полезны для кожи и шерсти
    (предрасположенность к атопии)

⚠️ Расчёт — отправная точка.
   Индивидуальная потребность может
   отличаться на ±30%.
```

### Стоп-лист (отдельная кнопка)

```
⛔ Никогда не давать:
  шоколад, лук/чеснок, виноград/изюм,
  ксилитол, авокадо, алкоголь,
  орехи макадамия, сырое дрожжевое тесто

⚠️ Нежелательно регулярно:
  сырые яичные белки, сырая рыба,
  варёные кости, чисто мясной рацион,
  беззерновые диеты без показаний

ℹ️ Пищевые аллергены определяются
  индивидуально через элиминационную диету
  у ветеринара. Бот не выдаёт запреты
  без диагноза.
```
