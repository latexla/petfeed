# Roadmap — PetFeed v1.0

## Принцип планирования

**Ship fast, learn fast.** Минимальный MVP за 6–8 недель, затем итерации на основе реального фидбека пользователей.

---

## Фаза 0 — Подготовка (Недели 1–2)

**Цель:** Всё готово до написания первой строки кода.

| Задача | Результат |
|---|---|
| Финализировать документацию | Все артефакты согласованы |
| Настроить окружение разработки | Python, PostgreSQL, Redis локально |
| Зарегистрировать Telegram-бота | @PetFeedBot, получить токен |
| Подключить DeepSeek API | API ключ, тест первого запроса |
| Провести кастодев (20+ человек) | Валидация гипотез, корректировка концепции |
| Договориться с 1 партнёром | Тестовая интеграция зоомагазина |

**Feature flags на этом этапе:** все OFF

---

### Неделя 5–6: Питание и напоминания

### Неделя 7–8: AI и заказы

## Фаза 3 — Масштабирование (Месяц 7–12)

**Цель:** 20,000 пользователей, мобильное приложение, федеральные партнёры.

| Задача | Сроки |
|---|---|
| Telegram Mini App (расширенный UI) | Месяц 7–8 |
| iOS приложение (React Native / Flutter) | Месяц 8–10 |
| Android приложение | Месяц 9–11 |
| Партнёрство с Petshop / 4лапы | Месяц 7 |
| PR в зоо-СМИ | Месяц 8 |
| B2B: белый лейбл для зоомагазинов | Месяц 10–12 |
| Горизонтальное масштабирование API | Месяц 9 |

**Метрики конца Фазы 3:**
- 20,000 активных пользователей
- Доход: 300,000–500,000 ₽/мес

## Общая шкала времени

```
Нед 1–2   [===] Подготовка

Мес 3     [====] Грызуны/птицы + трекер веса

Мес 12    [====] B2B + масштабирование
```

## Риски сроков

| Риск | Влияние на сроки | Митигация |
|---|---|---|
| Задержка интеграции партнёра | +1–2 недели | Начать с ручной ссылки без API |
| Проблемы с DeepSeek API | +1 неделя | Заготовить fallback на базу знаний |
| Медленный набор аудитории | Сдвиг Фазы 2 | Pivot на нишу (только экзотика) |
