# Анализ рынка — Pet Nutrition App

## 1. Идея проекта

**Продукт:** Приложение/чат-бот для помощи владельцам домашних животных в вопросах питания.  
**Старт:** Telegram-бот / Mini App  
**Следующий шаг:** Мобильное приложение (iOS + Android)  
**Целевые животные:** Кошки, собаки, грызуны, рептилии, птицы

---

## 2. Размер рынка

| Показатель | Значение |
|---|---|
| Глобальный рынок pet care (2025) | $181.9 млрд |
| Прогноз на 2033 | $283.7 млрд |
| CAGR 2026–2033 | 5.9% |
| Быстрорастущий регион | Азия-Тихоокеанский (6.3% CAGR) |

**Ключевой тренд:** "Pet humanization" — владельцы всё больше воспринимают питомцев как членов семьи и готовы платить за качество ухода.

## 3. Конкуренты

### Прямые конкуренты (приложения)

| Приложение | Рейтинг | Фокус | Слабые стороны |
|---|---|---|---|
| Pet Care Tracker (Nova Pet Health) | 4.6★ | Общий трекер | Нет фокуса на питании |
| Pawdi - Pet Food Scanner | — | Сканер штрих-кодов | Только готовый корм |
| Pet Food Calculator | — | Расчёт порций | Примитивный UX |
| Reptile Rocket | 4.4★ | Только рептилии | Узкая аудитория |
| PetDiary | — | Общий дневник | Не про питание |

### Вывод по конкурентам
- Большинство приложений либо **слишком общие** (здоровье в целом), либо **слишком узкие** (только один вид животных или только сканер корма)
- **Нет сильного игрока**, который закрывает питание для ВСЕХ видов домашних животных с персонализацией
- Интеграция с доставкой еды — **уникальная функция**, которой нет ни у кого из найденных конкурентов

---

## 4. Целевая аудитория

**Основной сегмент:** Городские владельцы домашних животных 18–40 лет

| Сегмент | Боль | Что ищут |
|---|---|---|
| Новички (впервые завели питомца) | Не знают что и сколько кормить | Простые чёткие инструкции |
| Осознанные владельцы | Хотят кормить правильно, но много противоречивой инфы | Персонализированные рекомендации |
| Владельцы экзотики (рептилии, грызуны) | Почти нет специализированных инструментов | Контент именно под их животное |
| Многопитомцевые | Сложно следить за каждым | Управление несколькими профилями |

---

## 5. Уникальное торговое предложение (УТП)

> **"Персональный нутрициолог для любого питомца — в одном боте"**

- Покрывает все виды домашних животных (не только кошки/собаки)
- Персонализация по возрасту, весу, породе, состоянию здоровья
- Рекомендации + прямая доставка подходящего корма
- Простой старт через Telegram (не нужно скачивать приложение)

---

## 6. Монетизация

### Фаза 1 — Telegram Bot / Mini App
- **Партнёрская программа с зоомагазинами** — % с каждого заказа через бота
- Бесплатный базовый функционал (привлечение аудитории)

### Фаза 2 — Мобильное приложение
- **Freemium подписка:** базовые рекомендации бесплатно, расширенный анализ и история — платно
- **B2B:** интеграция с ветеринарными клиниками (реферальная программа)
- **Партнёрство с производителями корма** — рекомендации брендов

### Ориентировочные цифры
| Метрика | Цель (6 мес.) |
|---|---|
| Пользователи бота | 5,000 |
| Конверсия в заказ через партнёра | 5–10% |
| Средний чек партнёра | 1,500–3,000 руб. |
| Комиссия | 5–15% |

## 7. Риски

| Риск | Вероятность | Митигация |
|---|---|---|
| Низкая retention (пользователи забывают о боте) | Высокая | Push-напоминания, геймификация (стрики кормления) |
| Сложность контента для экзотики | Средняя | База знаний + партнёрство с ветеринарами |
| Конкуренция с крупными платформами | Низкая на старте | Фокус на нише экзотики + Telegram |
| Доверие к рекомендациям | Средняя | Верификация с ветеринарами, источники |

---

## 8. Стратегия выхода на рынок

1. **Месяц 1–2:** Запуск Telegram-бота, кошки + собаки
2. **Месяц 3–4:** Добавление грызунов и птиц, первые партнёры по доставке
3. **Месяц 5–6:** Рептилии, аналитика retention, подготовка к мобильному приложению
4. **Месяц 7+:** MVP мобильного приложения, масштабирование партнёрств
