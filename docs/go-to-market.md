# Go-to-Market стратегия — PetFeed

## Общий подход

**Модель:** Community-led growth — сначала строим доверие в сообществах владельцев животных, потом масштабируем через партнёров.

**Принцип:** Не реклама, а ценность. Люди делятся ботом потому что он реально помогает, а не потому что мы платим за показы.

---

## Фазы выхода на рынок

### Фаза 0 — Подготовка (до запуска, 2–4 недели)

**Цель:** Создать базу для органического роста до первого пользователя.

| Действие | Детали |
|---|---|
| Создать Telegram-канал проекта | Рассказывать о разработке, делиться прогрессом |
| Собрать список сообществ | Группы владельцев кошек, собак, рептилий в Telegram и ВКонтакте |
| Кастодев (20+ интервью) | Опросить владельцев: чем кормят, какие проблемы, что ищут в интернете |
| Подготовить контент | 5–10 полезных постов о питании животных для посевов |
| Договориться с 1–2 партнёрами | Небольшие зоомагазины охотно идут на пилотную интеграцию |

---

### Фаза 1 — Запуск MVP (месяц 1–2)

**Цель:** 500 активных пользователей, первые заказы через партнёра.

#### Канал 1: Посевы в Telegram-сообществах

Целевые группы:
- Владельцы кошек и собак (крупные чаты 10k+)
- Владельцы рептилий и экзотики (нишевые, но очень лояльные)
- Группы по правильному питанию животных

Формат: не реклама, а полезный пост от лица участника + упоминание бота как инструмента.

**Бюджет:** 0–5,000 ₽ (договариваться с админами на бесплатное размещение или бартер)

#### Канал 2: ВКонтакте

- Группы владельцев животных (аудитория 18–45 лет активна в VK)
- Таргетированная реклама: интересы "животные", "зоомагазины", "ветеринария"
- Бюджет на тест: 3,000–5,000 ₽

#### Канал 3: Партнёрский трафик

- Зоомагазин-партнёр размещает ссылку на бот в своих соцсетях и рассылке
- Взаимовыгодно: магазин получает лояльных покупателей, мы — трафик

#### Канал 4: Виральность

Механика "Поделиться карточкой питомца":
- После создания профиля бот генерирует карточку с именем питомца и его рационом
- Пользователь делится в соцсетях → органический охват

---

### Фаза 2 — Рост (месяц 3–6)

**Цель:** 3,000–5,000 активных пользователей, стабильный доход от партнёрки.

#### Канал 5: SEO и контент

Статьи под поисковые запросы:
- "Чем кормить шиншиллу в домашних условиях"
- "Что нельзя давать кошке список"
- "Рацион для черепахи красноухой"

Каждая статья заканчивается CTA: "Составить рацион для вашего питомца бесплатно →"

**Инструмент:** Telegram Mini App или лендинг с переходом в бот.

#### Канал 6: Коллаборации с зооблогерами

- Микроинфлюенсеры (5k–50k подписчиков) охотно делают обзоры за бесплатный Premium
- Фокус: владельцы рептилий и экзотики (очень вовлечённая аудитория)
- Бюджет: 0 ₽ (бартер на Premium-доступ)

#### Канал 7: Ветеринарные клиники

- Партнёрство: клиника рекомендует бот своим пациентам
- Взамен: специальный раздел "Рекомендовано ветеринаром" в профиле
- Первые 3–5 клиник закрываются личными переговорами

---

### Фаза 3 — Масштабирование (месяц 7–12)

**Цель:** 10,000+ пользователей, запуск мобильного приложения.

| Действие | Детали |
|---|---|
| Запуск iOS/Android приложения | На базе накопленной аудитории Telegram |
| Расширение партнёрской сети | Федеральные зоомагазины (Petshop, 4лапы) |
| PR в зоо-СМИ | Статьи в Zoovet.ru, Питомцы Mail.ru |
| Программа лояльности | Бонусы за приглашение друзей |

## Целевые метрики по каналам

| Канал | CAC (стоимость привлечения) | Ожидаемый объём/мес |
|---|---|---|
| Посевы Telegram | 0–30 ₽ | 200–500 пользователей |
| Таргет ВКонтакте | 50–150 ₽ | 100–300 пользователей |
| Партнёрский трафик | 0 ₽ | 100–200 пользователей |
| Виральность | 0 ₽ | 50–150 пользователей |
| SEO (с 3-го мес.) | 0 ₽ | 100–500 пользователей |
| Блогеры | 0 ₽ (бартер) | 200–1000 пользователей |

## Позиционирование в коммуникации

### Ключевое сообщение
> "Правильное питание для любого питомца — за 2 минуты"

### Для владельцев собак и кошек
> "Перестань гуглить — получи точный рацион для своего питомца прямо в Telegram"

### Для владельцев экзотики
> "Наконец-то инструмент, который знает что едят рептилии, грызуны и птицы"

### Для партнёров
> "Рекомендуй нужный корм в момент, когда пользователь уже хочет его купить"
