# Бизнес-модель — PetFeed

## Общая стратегия

**Подход:** Freemium + партнёрская программа  
**Принцип:** Сначала аудитория — потом монетизация  
**Старт:** Бесплатный бот с партнёрской комиссией с первого дня

---

## Потоки дохода

### Поток 1 — Партнёрская программа (старт)
Интеграция с зоомагазинами и сервисами доставки.

- Пользователь получает рекомендацию корма → нажимает "Заказать"
- Заказ оформляется через партнёра
- PetFeed получает комиссию **5–15%** с каждого заказа

**Потенциальные партнёры:** Petshop, Зоозавр, Wildberries (зоотовары), 4лапы, локальные зоомагазины

| Показатель | Значение |
|---|---|
| Средний чек заказа | 1,500–3,000 ₽ |
| Комиссия | 5–15% |
| Доход с заказа | 75–450 ₽ |

---

### Поток 2 — Подписка Premium (Фаза 2)

| Тариф | Цена | Что включено |
|---|---|---|
| Бесплатный | 0 ₽ | 1 питомец, базовые рекомендации |
| Premium | 299 ₽/мес | До 5 питомцев, аналитика, сканер корма, AI-ассистент |
| Family | 499 ₽/мес | Безлимит питомцев, приоритетная поддержка |

---

### Поток 3 — B2B (Фаза 3)

- **Ветеринарные клиники** — реферальная программа, виджет на сайт клиники
- **Производители корма** — платное размещение в рекомендациях (с пометкой "реклама")
- **Зоомагазины** — белый лейбл бота под брендом магазина

## Unit-экономика

### Сценарий: 1,000 активных пользователей

| Параметр | Значение |
|---|---|
| Активных пользователей | 1,000 |
| Конверсия в заказ (5%) | 50 заказов/мес |
| Средний чек | 2,000 ₽ |

| Параметр | Значение |
|---|---|
| Активных пользователей | 10,000 |
| Партнёрские заказы (5%) | 500/мес → **100,000 ₽** |
| Premium подписчики (3%) | 300 × 299 ₽ → **89,700 ₽** |
| **Итого в месяц** | **~190,000 ₽** |

## Стратегия роста аудитории

1. **Органика:** Посевы в Telegram-каналах о животных (много бесплатных)
2. **Виральность:** "Поделись профилем питомца" — карточка с рационом
3. **SEO/контент:** Статьи "Чем кормить [животное]" → в бота
4. **Партнёры:** Зоомагазины продвигают бота своей аудитории
5. **Сообщества:** ВКонтакте и Telegram-группы владельцев экзотики
