# User Stories — PetFeed

## Роли пользователей

| Роль | Описание |
|---|---|
| **Владелец питомца** | Основной пользователь, имеет одного или нескольких питомцев |
| **Новый пользователь** | Впервые запускает бота, ещё не создал профиль |
| **Партнёр (зоомагазин)** | Поставщик корма, интегрированный в платформу |
| **Администратор** | Управляет контентом и базой знаний |

---

## Эпик 1: Онбординг и профиль питомца

### US-001: Регистрация питомца
```
Как новый пользователь,
я хочу создать профиль своего питомца (вид, порода, возраст, вес),
чтобы получать персональные рекомендации по питанию именно для моего животного.
```

### US-002: Несколько питомцев
```
Как владелец питомца,
я хочу добавить несколько питомцев в один аккаунт,
чтобы управлять рационом каждого из них в одном месте.
```

### US-003: Редактирование профиля
```
Как владелец питомца,
я хочу обновить данные питомца (вес, возраст, состояние здоровья),
чтобы рекомендации оставались актуальными по мере роста животного.
```

---

## Эпик 2: Рекомендации по питанию

### US-004: Получение рациона
```
Как владелец питомца,
я хочу получить персональный рацион питания для своего животного,
чтобы кормить его правильно и не тратить время на поиск информации в интернете.
```

### US-005: Стоп-список продуктов
```
Как владелец питомца,
я хочу видеть список запрещённых продуктов для своего животного,
чтобы случайно не навредить питомцу опасной едой.
```

### US-006: Расчёт порций
```
Как владелец питомца,
я хочу узнать точный объём и частоту кормления,
чтобы питомец не переедал и не испытывал голод.
```

### US-007: Рекомендации по готовому корму
```
Как владелец питомца,
я хочу получить рекомендации по подходящим маркам готового корма,
чтобы не тратить время на самостоятельное изучение состава.
```

---

## Эпик 3: Дневник кормления и напоминания

### US-008: Напоминание о кормлении
```
Как владелец питомца,
я хочу настроить напоминания о времени кормления,
чтобы не пропускать приёмы пищи питомца при загруженном расписании.
```

### US-009: Журнал кормлений
```
Как владелец питомца,
я хочу отмечать каждое кормление в журнале,
чтобы отслеживать соблюдение рациона и видеть историю питания.
```

### US-010: Отслеживание веса
```
Как владелец питомца,
я хочу регулярно вносить вес питомца и видеть динамику,
чтобы своевременно замечать отклонения и корректировать питание.
```

---

## Эпик 4: Заказ корма

### US-011: Заказ рекомендованного корма
```
Как владелец питомца,
я хочу заказать рекомендованный корм напрямую из бота одной кнопкой,
чтобы не искать его самостоятельно и сэкономить время.
```

### US-012: Выбор партнёра доставки
```
Как владелец питомца,
я хочу выбрать из нескольких магазинов-партнёров,
чтобы сравнить цены и выбрать удобный вариант доставки.
```

### US-013: История заказов
```
Как владелец питомца,
я хочу видеть историю своих заказов корма,
чтобы быстро повторить предыдущий заказ без повторного поиска.
```

---

## Эпик 5: AI-ассистент

### US-014: Вопрос-ответ по питанию
```
Как владелец питомца,
я хочу задать свободный вопрос о питании своего животного,
чтобы получить быстрый ответ без визита к ветеринару по несрочным вопросам.
```

### US-015: Анализ состава корма
```
Как владелец питомца,
я хочу отправить фото или название корма и получить его анализ,
чтобы понять, подходит ли он моему питомцу.
```

---

## Эпик 6: Администрирование

### US-016: Управление базой знаний
```
Как администратор,
я хочу добавлять и редактировать данные о питании для разных видов животных,
чтобы информация в боте оставалась актуальной и достоверной.
```

### US-017: Подключение партнёра
```
Как партнёр (зоомагазин),
я хочу интегрировать свой каталог товаров в платформу,
чтобы получать заказы от пользователей бота.
```
