# База знаний питания — PetFeed v1.0

## Структура и правила наполнения

### Принципы
- Только верифицированные данные из ветеринарных источников
- Каждая запись имеет ссылку на источник
- Обновление — только через Admin Panel с указанием автора
- При сомнении — флаг "уточните у ветеринара", не додумывать

### Источники данных
1. ASPCA Animal Poison Control Center (aspca.org)
2. VCA Hospitals (vcahospitals.com)
3. Merck Veterinary Manual
4. Российские ветеринарные клиники (партнёрская верификация)

## Формулы расчёта рациона

### RER (Resting Energy Requirement) — базовая формула
```
RER = 70 × (weight_kg ^ 0.75)
```

### Коэффициенты по цели
| Цель | Коэффициент |
|---|---|
| maintain (поддержание) | × 1.0 |
| lose (похудение) | × 0.8 |
| gain (набор веса) | × 1.2 |
| growth (рост, < 12 мес) | × 1.5 |
| growth (рост, 12–18 мес) | × 1.2 |

### Частота кормлений по возрасту
| Возраст | Кошки/Собаки | Грызуны | Рептилии | Птицы |
|---|---|---|---|---|
| < 6 мес | 4 раза/день | постоянный доступ | 1 раз/день | постоянный доступ |
| 6–12 мес | 3 раза/день | постоянный доступ | 1 раз/2 дня | постоянный доступ |
| > 12 мес | 2 раза/день | постоянный доступ | 1 раз/3 дня | постоянный доступ |

### Нормы веса по видам (для предупреждений)
| Вид | Мин. вес (кг) | Макс. вес (кг) | Примечание |
|---|---|---|---|
| Кошка | 2.5 | 8.0 | Зависит от породы |
| Собака | 1.5 | 80.0 | Зависит от породы |
| Хомяк | 0.08 | 0.15 | |
| Крыса | 0.15 | 0.50 | |
| Морская свинка | 0.70 | 1.20 | |
| Шиншилла | 0.40 | 0.70 | |
| Кролик | 1.0 | 6.0 | Зависит от породы |
| Черепаха (красноухая) | 0.10 | 3.0 | |
| Попугай (волнистый) | 0.03 | 0.05 | |

## Правила для Admin Panel

### Добавление новой записи
1. Указать `species` из допустимого enum
2. Указать `product_name` — точное название на русском
3. Установить `is_allowed`: TRUE / FALSE
4. Если `is_allowed = FALSE` — обязательно указать `danger_level` и `reason`
5. Указать `source_url` — ссылка на ветеринарный источник
6. При сохранении — инвалидировать Redis-кэш рационов для данного `species`

### Уровни опасности
| Уровень | Отображение в боте | Когда использовать |
|---|---|---|
| `moderate` | ⚠️ С осторожностью | Небольшой вред при больших дозах |
| `dangerous` | 🚫 Опасно | Вред при любом количестве |
| `fatal` | ☠️ Смертельно | Летальный исход возможен |
